Anurag AI Assistant

Anurag AI Assistant is a full-stack, AI-powered chatbot built using Python, Flask, and Hugging Face Router (OpenAI-compatible LLM API).
It provides intelligent, domain-specific responses with multiple learning modes and a responsive web interface.
